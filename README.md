# Order Management API

## 📌 Descrição
API para gerenciamento de pedidos, simulando regras reais de negócio.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- Docker
- Clean Architecture
- Design Patterns

## ▶️ Como rodar

```bash
docker-compose up -d
./mvnw spring-boot:run
```

## 📚 Endpoints
- POST /orders
- GET /products

## 🧠 Conceitos aplicados
- Separação por camadas
- Regras de negócio no domínio
- Use Cases
